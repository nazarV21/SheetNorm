Конвертация Excel

Обработайте один файл, сразу несколько файлов или целую папку. Все задачи продолжат выполняться в фоне.

1. Выберите файлы

Перетащите сюда один или несколько файлов .xlsx/.xls

Каждый файл станет отдельной фоновой задачей. Для одного файла откроется его карточка, для нескольких — страница пакета.

Универсальный вариант

Работает почти во всех браузерах. После обработки можно скачать готовые файлы одним ZIP.

Сохранить результаты рядом

Для Chrome/Edge. SheetNorm создаст подпапку SheetNorm_results в выбранной папке и будет сохранять туда готовые файлы.

Доступность зависит от браузера и разрешения на запись.
