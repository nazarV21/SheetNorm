# AI-помощник по инструкции обработки Excel

AI-помощник предназначен для пользователей, которые не знают технических параметров преобразования таблиц: `header_rows`, `melt`, `id_columns`, JSON-правила и другие внутренние настройки.

## Основной принцип

Пользователь взаимодействует только с естественным языком:

> Нужно сделать фильтр по филиалам и посчитать итоги продаж для выбранного филиала.

Система выполняет следующие действия:

1. анализирует Excel-файл;
2. определяет вероятные служебные строки, заголовки и начало данных;
3. улучшает инструкцию пользователя;
4. автоматически строит внутреннее JSON-правило;
5. применяет правило и показывает предпросмотр итоговой таблицы;
6. при изменении инструкции повторно анализирует файл и пересобирает правило.

## Почему JSON скрыт

JSON-правило является внутренним техническим представлением. Оно нужно алгоритму, но не даёт пользы обычному пользователю. Поэтому в интерфейсе показывается не JSON, а человекочитаемое описание:

- будет включён автофильтр;
- будет добавлена строка итогов;
- будет создан лист с итогами;
- таблица будет приведена к плоскому формату.

## Обучение на ошибках

Если пользователь меняет предложенную инструкцию, система сохраняет запись в `instruction_feedback.json`:

- исходная инструкция пользователя;
- инструкция, предложенная системой;
- исправленная инструкция пользователя;
- новая инструкция после повторного анализа;
- признаки структуры Excel-файла;
- автоматически сгенерированное правило;
- колонки предпросмотра результата.

При следующих похожих файлах эти записи используются как примеры прошлых исправлений. Это не fine-tuning весов модели, а обучение через feedback memory и few-shot-контекст.

## Сценарий с филиалами и итогами

Если пользователь пишет, что нужно сделать фильтр по филиалам и посчитать продажи, система старается автоматически добавить в итоговый Excel:

- автофильтр по таблице;
- закрепление строки заголовков;
- строку итогов с формулами `SUBTOTAL`, которые пересчитываются после выбора филиала в фильтре;
- отдельный лист `Итоги` с суммами по филиалам, если это требуется инструкцией.

## Пользовательский цикл

1. Загрузить Excel.
2. Написать инструкцию.
3. Посмотреть предпросмотр результата.
4. Если результат не подходит — изменить инструкцию.
5. Нажать «Обновить анализ и предпросмотр».
6. Сохранить удачную инструкцию как типовой шаблон.
