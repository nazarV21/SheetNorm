Пример промпта для правила "БВ-2.3а: Преобразование в плоскую таблицу"

Это сложное преобразование из многоуровневой структуры в плоскую таблицу. 

Структура исходной таблицы:
- Заголовки находятся на строках 2-4
- "№ п.п." и "Наименование работ" - вертикальные столбцы
- "Группы подъемных агрегатов" (до 50 т, от 50 до 80 т, свыше 100 т) - горизонтальные заголовки
- Значения в ячейках - нормы времени

Структура итоговой таблицы должна быть:
1. Блок-вставки по КРС (всегда "БВ-2.3а")
2. Наименование работ
3. Группы подъемных агрегатов
4. Норма времени, час

Промпт для ИИ:
---
Преобразовать многоуровневую таблицу в плоскую структуру.

Исходная таблица имеет структуру:
- В первом столбце находятся номера (№ п.п.) и наименования работ
- Группы подъемных агрегатов ("до 50 т", "от 50 до 80 т", "свыше 100 т" и т.д.) - это горизонтальные заголовки
- В ячейках находятся нормы времени

Задача:
1. Добавить первый столбец "Блок-вставки по КРС" со значением "БВ-2.3а" для всех строк
2. Развернуть кросс-таблицу в плоскую: каждая комбинация (наименование работы + группа) должна стать отдельной строкой
3. Столбцы итоговой таблицы:
   - "Блок-вставки по КРС" = "БВ-2.3а"
   - "Наименование работ" = берется из первого столбца исходной таблицы
   - "Группы подъемных агрегатов" = берется из названия группы (столбец с нормами)
   - "Норма времени, час" = значение нормы времени из соответствующей ячейки

Правила обработки:
- Пропускать пустые ячейки
- Если наименование работы занимает несколько строк (например, "Монтаж подъемного агрегата" с подпунктами "с оттяжками", "без оттяжек"), правильно их объединять
- Значения норм времени должны быть числовыми
